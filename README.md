# 🏠 LivEra – Smart Student Housing Platform

## 📌 Overview

LivEra is a smart student housing platform designed to help students find verified rental properties near educational institutions. The platform provides trusted listings, smart search filters, location-based discovery, and a seamless user experience while eliminating broker dependency.



## 🚨 Problem Statement

Students often face several challenges while searching for accommodation:

- Fake or misleading listings
- High broker charges
- Lack of trust and verification
- Time-consuming search process
- Poor filtering and location discovery



## 💡 Our Solution

LivEra solves these problems by providing:

- ✅ Verified property listings
- ✅ Smart search filters
- ✅ Location-based property discovery
- ✅ Transparent information
- ✅ Broker-free housing search



## ✨ Features
